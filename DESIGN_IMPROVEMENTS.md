## 📱 Breakpoints المستخدمة

```css
/* Mobile First */
default: < 640px   (الموبايل)
sm: >= 640px       (الهاتف الكبير)
md: >= 768px       (التابلت)
lg: >= 1024px      (اللابتوب)
xl: >= 1280px      (الشاشة الكبيرة)
```

## 🎯 الكلاسات الرئيسية المستخدمة

### من `index.css`:
- `card` - بطاقة بيضاء مع حدود
- `card-hover` - تأثير عند المرور
- `btn`, `btn-primary`, `btn-secondary`, etc. - أزرار
- `input` - حقول الإدخال
- `label` - التسميات
- `badge` - شارات الحالة
- `modal-overlay`, `modal-content` - النوافذ المنبثقة
- `empty-state` - الحالة الفارغة
- `stat-icon` - أيقونات الإحصائيات

### Responsive Classes:
- `flex-col sm:flex-row` - عمودي على الموبايل، أفقي على الأكبر
- `grid-cols-1 sm:grid-cols-2 lg:grid-cols-3` - عدد أعمدة متغير
- `w-full sm:w-auto` - عرض كامل على الموبايل، تلقائي على الأكبر
- `p-4 sm:p-6` - Padding متغير حسب الحجم

## 🔄 التأقلم التلقائي

### الموبايل → اللابتوب:
1. **Grid**: 1 col → 2 cols → 3 cols
2. **Flex**: column → row
3. **Buttons**: full width → auto width
4. **Spacing**: compact → comfortable
5. **Modals**: full screen → centered with max-width

### اللابتوب → الموبايل:
- العكس بشكل تلقائي باستخدام Tailwind breakpoints
